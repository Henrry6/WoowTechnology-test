## ¿Por qué elegiste esas librerías?

Elegí Node.js + Express por ser liviano y fácil de organizar para APIs. PostgreSQL porque es estable y maneja bien relaciones entre datos. bcryptjs para encriptar contraseñas seguro y simple. Axios facilita la comunicación frontend-backend. React + Vite porque hacen el desarrollo rápido y modular. Ant Design me ahorra tiempo con componentes listos y consistentes.

## ¿Qué desafíos enfrentaste?

Tuve problemas con los seeds hasta ajustar bien las tablas y columnas, y al inicio los errores del backend se mostraban en HTML, lo que no es ideal para una API.

## ¿Qué mejorarías con más tiempo?

Agregar validaciones más completas, logging y tests para tener un backend más sólido y seguro.

## ¿Cómo escalarías esta solución?

Separando servicios en microservicios, usando caching, balanceadores de carga y una base de datos optimizada para consultas pesadas.
